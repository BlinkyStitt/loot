Loot Splitter
=============

Setup
-----

    $ . setup.source_me

Example
-------

    $ python loot.py --cp 100 --ep 0 --sp 700 --gp 2000 --pp 18 -n 4
    split 4 pp with 2 leftover...
    dividing 2 pp...
    dividing 1 pp...
    dividing 2000 gp...
    dividing 1999 gp...
    dividing 1998 gp...
    dividing 1997 gp...
    dividing 1996 gp...
    dividing 1995 gp...
    dividing 1994 gp...
    dividing 1993 gp...
    dividing 1992 gp...
    dividing 1991 gp...
    dividing 1990 gp...
    dividing 1989 gp...
    dividing 1988 gp...
    dividing 1987 gp...
    dividing 1986 gp...
    dividing 1985 gp...
    dividing 1984 gp...
    dividing 1983 gp...
    dividing 1982 gp...
    dividing 1981 gp...
    split 495 gp with 0 leftover...
    split 175 sp with 0 leftover...
    split 25 cp with 0 leftover...
